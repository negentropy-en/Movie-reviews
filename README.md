# Movie-reviews

## Description

Movies-Reviews is a Flask web application where users can:
- Create an account and log in to the application
- Post movie-reviews
- Edit and delete their own reviews
- Browse everyone's reviews
- Search reviews by keyword
- User page shows how many movie-reviews the user has added and a list of the movie-reviews the user has added.
- Choose one or more ratings for the movie-review (e.g., bad, average, great).
- Give a comment and a rating to the movie-review. The comments and the average rating are displayed in the movie-review.

Here, the primary data item is the movie-review and the secondary data item is the comment on the movie-review.

## Technical requirements
This project was built to satisfy the course requirements:
- Python (version 3.10)
- Flask
- SQLite for data storage
- Direct SQL queries with 'sqlite3' (no ORM)
- Git and GitHub used for version control
- HTML templates + CSS only
- No JavaScript
- No external Python libraries other than Flask and what Flask already depends on (Werkzeug etc.)

## Language of the project
The language of the application's user interface and README.md file is English. The language of the technical implementation of the application (code, database, commit messages) is English.
